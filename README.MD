# Students CRUD application

Simple crud app with auth for learning vue.js

## Installation

### backend

cd CrudStudentsApp\backend

```bash
make env-copy
```

```bash
make init
```

### frontend

cd CrudStudentsApp\frontend
```bash
make env-copy
```
```bash
make init
```

## Tests and linters

### backend
cd CrudStudentsApp\backend

stat analise
```bash
make stan
```

code style
```bash
make cs-fixer
```

### frontend
cd CrudStudentsApp\frontend

unit tests
```bash
make front-u-test
```

linter
```bash
make front-linter
```
